¿Qué es una computadora?

  • Hardware
  • Software

Hardware:

  • CPU
  • Memoria (RAM)
  • Almacenamiento
  • Periféricos

CPU

Unidad central de procesamiento o procesador

“Es el encargado de ejecutar todos los programas, desde el sistema operativo hasta las aplicaciones de usuario; solo ejecuta instrucciones en lenguaje binario, realizando operaciones aritméticas y lógicas simples, tales como sumar, restar, multiplicar, dividir, las lógicas binarias y accesos a memoria.” [1]

Algunas características que definen a un CPU

  • Arquitectura (x86, x86_64, ARM, PowerPC)
  • Velocidad (Mhz, Ghz)
  • Cantidad de cores y threads
  • Set de instrucciones

Memoria

Memoria RAM (Ramdom Access Memory)

“En la RAM se cargan todas las instrucciones que ejecuta la unidad central de procesamiento (CPU) y otras unidades del computador, además de contener los datos que manipulan los distintos programas.” [2]

Almacenamiento

NO confundir con memoria

  • Unidades internas (discos hdd, ssd, flash)
  • Unidades externas (discos usb, pendrives)
  • Almacenamiento por red local o en la nube

Los programas se guardan y pueden guardar información en el almacenamiento.

Periféricos

Son para interactuar con la computadora

  • Teclado, mouse, touch (Entrada)
  • Monitor, sonido, impresora, luces (Salida)

¿Qué es la programación?

Básicamente es escribir programas de computadora.

Programa:

“Un programa es un conjunto de instrucciones, ejecutables sobre una computadora, que permite cumplir una function específica.”
[3]

Ejemplo de un programa

Pseudocódigo


								# Lee el nombre desde el teclado
								imprimir_en_pantalla "Por favor ingrese su nombre"
								leer_desde_teclado (nombre)

								# Realiza el saludo con el nombre
								imprimir_en_pantalla "¡Hola $nombre!"

Ejecutar programa: 


								ejecutar programa.exe 
								Por favor, ingrese su nombre: Cristian
								¡Hola Cristian!

En Python


								# Lee el nombre desde el teclado
								nombre = input("Por favor, ingrese su nombre: ")
								# Realiza el saludo con el nombre
								print ("¡Hola " + nombre + "!")

En C


								#include "stdio.h"
								int main( ) {
									char str[64];
									printf( "Por favor, ingrese su nombre: ");
									// Lee el nombre desde el teclado
									scanf("%s", str);
									// Realiza el saludo con el nombre
									printf( "\n¡Hola %s!", str);
									return 0; 
								}

Lenguajes de programación

Lenguaje: Es un lenguaje formal que le proporciona a una persona la capacidad de escribir una serie de instrucciones en forma de algoritmos para ser ejecutados en una computadora.

Clasificación de lenguajes

  • Nivel: De alto nivel | bajo nivel
  • Paradigmas: Orientado a objetos, procedural, eventos, reactiva... etc
  • Propósito: Propósito general y específicos
  • Compilados e interpretados
  • Tipados no tipados

Librerías

Una librería (library) es un conjunto de implementaciones funcionales, codificadas en un lenguaje de programación, que ofrece una interfaz bien definida para la funcionalidad que se invoca.

Pueden extender las capacidades estándar del lenguaje y/o ofrecer al desarrollador herramientas útiles.

Algoritmos

“Algoritmo es la especificación rigurosa de la secuencia de pasos (instrucciones) a realizar sobre un autómata [computadora] para alcanzar un resultado deseado en un tiempo finito [comienza y termina].”
[3]

"Una secuencia de pasos a seguir. Como si fuera una receta."

Algoritmos (cont)

Todos los lenguajes de programación tienen un conjunto mínimo de instrucciones... este conjunto debe contener estructuras de control para la asignación, decisión e iteración.

A continuación veremos el comportamiento de los mas básicos.

Asignación [=]

Asigna un valor a una "variable" o constante para luego usar en la ejecución del programa.

Una variable es un lugar en la memoria (RAM) donde podremos almacenar valores temporalmente.

El tiempo de vida o alcance de una variable se verá mas adelante en el curso.


								# Ejemplo de asignación
								nombre= "Cristian"
								edad= 35

Secuencia

Una sucesión de instrucciones. Se ejecutan en orden de arriba hacia abajo.

Secuencia en Python


								print("Instrucción 1")
								print("Instrucción 2")
								print("Instrucción 3")
								print("Instrucción 4")

El resultado:


								Instrucción 1
								Instrucción 2
								Instrucción 3
								Instrucción 4

El orden siempre será desde arriba hacia abajo.

Decisión o selección (if)

Un algoritmo que representa un problema real sería imposible que sean solo instrucciones secuenciales. Es necesario tomar decisiones en función de los datos del problema.

El flujo se verá modificado.

Decisión (if) en Python

Primer caso


								if llueve:
									print("Tomar paraguas") # Instrucción 1
								print ("Salir al trabajo") # Instrucción 2

Si llueve la salida es:


									Tomar paraguas  
									Salir al trabajo

Si NO llueve la salida es:


								Salir al trabajo

Decisión (if) en Python [p2]

Segundo caso


								if llueve:
									print("Tomar paraguas") # Instrucción 1
								else:
									print("Abrir ventanas de la casa") # Instrucción 2
								print ("Salir al trabajo")  # Instrucción 3

Si llueve la salida es:


									Tomar paraguas   # Instrucción 1 
									Salir al trabajo # Instrucción 3

Si NO llueve la salida es:


									Abrir ventanas de la casa # Instrucción 2
									Salir al trabajo # Instrucción 3

Decisión o selectiva múltiple [case o switch]

Es una extensión de la estructura básica de decisión, para el caso de que las alternativas sean mas de dos:

Si mas de una condición es verdadera puede haber distintas formas de elegir cual caso tomar.

Case en Python

Agregado en Python 3.10


								match numero:
									case 0:
										print ("Cero")
									case 1:
										print ("Uno")
									case 2:
										print ("Dos")
									case _:
										print ("Es otro número")

Si el número es 0, 1 o 2 el resultado será:


								Cero, Uno o Dos (respectivamente)

Si es otro número, el resultado será


								Es otro número

Case en Python (elif)

Para usar en versiones anteriores a 3.10

Se puede usar con elif anidados


								numero= 4 # Asignamos el valor
								if numero == 0:
									print ("Cero")
								elif numero == 1:
									print ("Uno")
								elif numero == 2
									print ("Dos")
								else:
									print ("Es otro número")

Si corremos el programa, el resultado será


								Es otro número

Iteración [while, for]

La iteración es la capacidad de ejecutar un bloque de sentencias de forma repetida.

Se puede entender como mientras (while) se cumpla la condición se repite.

ó ejecutar n veces (for) un bloque de instrucciones en caso que se sepa de antemano.

Iteración en Python

Mientras (while)


								
								while llueve:
									print("Espero a que pare") 							
								print("Voy a clase")

La salida es:


									Espero a que pare
									Espero a que pare
									Espero a que pare
									Espero a que pare
									Espero a que pare
									Espero a que pare
									Espero a que pare
									Espero a que pare
									...
									Voy a clase

¡Cuidado pueden hacerse loops infinitos!

Iteración en Python

Mientras (while) - Ejemplo 2


								# Tengo que contar hasta 6
								miContador = 0
								while miContador < 6:
									print(miContador) 	# Voy contando el número
									miContador += 1			# Incremento el contador
								print("Terminé de contar")

La salida es:


									0
									1
									2
									3
									4
									5
									Terminé de contar

Iteración en Python

Repetición (for)


								for x in range(6):
									print(x)
								print("Terminé de contar")

La salida es:


									0
									1
									2
									3
									4
									5
									Terminé de contar

Iteración en Python [p2]

Iteración en una lista* (for)

La salida es:


									manzana

Sentencia break

A veces no sabemos que es momento de terminar un bucle hasta que llegas a la mitad del cuerpo. En ese caso podemos utilizar la sentencia break para saltar hacia afuera del bucle.

Supongamos que queremos tomar la entrada del usuario hasta que se escriba 'listo'


								while True:
									linea = input('Su texto: ')
									if linea == 'listo':
										break # Rompe el while
									print(linea)
								print('¡Listo!')

Como la condición del bucle es True siempre será verdadera y por lo tanto nunca corta. Salvo cuando se ejecuta la sentencia break.

Documentación de un algoritmo

  • Un programa bien documentado será mas fácil de leer y mantener.
  • La mayoría de lenguajes proveen mecanismos de documentación (comentarios)
  • Es recomendable que los identificadores sean auto explicativos

Corrección y verificación de algoritmos

  • El programa debe resolver los problemas planteados en la especificación
  • Probar con un amplio juego de datos reales
  • Los test con los datos para probar pueden escribirse antes que el programa
  • CI/CD

Eficiencia de un algoritmo

  • Puede haber muchas soluciones a un mismo problema
  • ...y pueden ser todas correctas.
  • Algunas soluciones pueden ser mas eficientes que otras en recursos de computación (tiempo, memoria, etc)

Las partes de un programa

Como se mencionó antes, los componentes básicos de un programa son:

  • Instrucciones
  • Datos

Estructuras y datos

Las partes de un programa (cont)

  • Instrucciones: Son las operaciones que ejecutará la computadora al interpretar el programa.
    • Asignación
    • Decisión
    • Iteración
  • Datos: Son valores de información que necesitamos disponer y en ocasiones transformar para ejecutar una función del programa
    • Constantes: Se definen al comienzo y no cambian durante la ejecución del programa
    • Variables: Se definen antes de usarlas y pueden cambiar durante la ejecución del programa.

    • Para cada uno de los datos, se definen identificadores (nombre). Para poder referirnos a esos datos en cualquier parte del programa.
